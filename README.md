## ![kite logo](https://github.com/HYbutterfly/kite/wiki/image/kite_logo.png)

Kite is a simple 2d game engine for Lua

## Build & Test (for windows mingw)
1. install mingw64 and gcc
2. cd kite and make
3. ./kite.exe examples/helloworld
